# Nhập Lý do từ chối bằng File Excel

## 1. Mục tiêu

* Nhập hàng loạt danh sách các lý do từ chối đã được chuẩn hóa từ file Excel vào hệ thống cùng lúc.
* Đảm bảo tính nhất quán của dữ liệu danh mục, tránh việc sai lệch do nhập tay nhiều lần.

## 2. Khi nào dùng?

* Khi mới thiết lập hệ thống và cần đổ toàn bộ bộ danh mục lý do từ chối theo quy định của tổng công ty.
* Khi thực hiện cập nhật quy mô lớn hoặc thay đổi cơ cấu danh mục phân loại lý do.

## 3. Các bước thực hiện chi tiết

### Bước 1: Truy cập tính năng Import:&#x20;

* Tại thanh công cụ màn hình chính, chọn biểu tượng ... (menu mở rộng) cạnh nút Cột, sau đó chọn dòng Nhập dữ liệu (Import).

### Bước 2: Tải file lên hệ thống:&#x20;

* Tại màn hình cấu hình, chọn bảng dữ liệu đích (Ví dụ: MBW ATS).
  * Thực hiện tải tệp Excel đã điền sẵn dữ liệu lý do từ máy tính cá nhân lên hệ thống.

### Bước 3: Kiểm tra và Khớp dữ liệu:&#x20;

* Hệ thống sẽ hiển thị bảng xem trước. Bạn cần đảm bảo các cột trong file Excel (Tên lý do, Phân loại, Trạng thái) đã khớp hoàn toàn với các trường thông tin tương ứng trên hệ thống.

### Bước 4: Hoàn tất nhập liệu:&#x20;

* Kiểm tra lần cuối và chọn xác nhận để đưa toàn bộ danh sách vào bảng quản lý chính.

## 4. Kết quả mong đợi

* Danh sách tại màn hình chính được cập nhật đầy đủ các bản ghi mới từ file Excel.
* Thông tin đồng bộ, không bị lỗi font chữ hoặc sai lệch cột thông tin.

## 5. Lưu ý quan trọng

* Sử dụng file mẫu: Để tránh lỗi định dạng, bạn nên thực hiện thao tác Xuất dữ liệu (Export) trước để lấy file Excel mẫu có sẵn các tiêu đề trường (ID Lý do, SyncID, Trạng thái...) của hệ thống.
* Phân loại chính xác: Đảm bảo cột "Phân loại" trong file Excel trùng khớp với các nhóm đã được định nghĩa trên hệ thống để tránh việc tạo ra các nhóm rác.
